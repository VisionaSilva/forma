FAQ

What is Forma? ▼
Forma is a contract-first CSS design system that prioritizes quality reproducibility. The same components work in every theme without modification.
How do I use Forma? ▼
Link one of the compiled CSS files (forma-dusk.css, forma-graphite.css, forma-ivory.css) and use the semantic class names in your HTML.
Can I create my own theme? ▼
Yes! Create a new folder in /themes/ with a theme.json that defines all the semantic color tokens. Run validate.js to ensure it matches the contract.
