Frequently Asked Questions

What makes Forma different from other design systems? ▼
Most design systems focus on components. Forma focuses on quality reproducibility. We use semantic tokens and a contract-based approach. The same HTML works in every theme without modification. That's the difference.
Do I need to buy a license? ▼
No. Forma is MIT licensed and completely open source. Use it freely in any project, commercial or personal. The Professional license is optional and unlocks custom theme generation and priority support.
Can I create my own theme? ▼
Absolutely. Create a new folder in /themes/ with a theme.json file. Define all the semantic color tokens, and run validate.js to ensure it matches the contract. All components will automatically adapt to your new theme.
How do I integrate Forma into my project? ▼
Link one of the compiled CSS files (forma-dusk.css, forma-graphite.css, or forma-ivory.css) in your HTML head. Use the semantic class names in your markup. That's it.
Is Forma production-ready? ▼
Yes. Forma includes 12 component categories covering all common UI patterns. The three included themes (Dusk, Graphite, Ivory) are production-ready and battle-tested.
